## 🛠️ ساختار هدف

```
deepchat2/
├── apps/
│   ├── frontend/     # Next.js با تم Mario
│   ├── backend/      # NestJS میکروسرویس
│   ├── agent/        # مدیریت AI
│   └── cli/          # ابزارهای CLI
├── packages/
│   ├── ui/           # کامپوننت‌های Mario
│   ├── config/       # تنظیمات متمرکز
│   └── lib/          # ابزارهای کمکی
└── docs/             # مستندات
```
